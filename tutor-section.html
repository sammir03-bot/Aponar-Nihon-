AI টিউটর আপনার নিহোন
অনলাইন

আপনার ব্যক্তিগত জাপানি টিউটর

AI দিয়ে বাংলায় জাপানি ভাষা শিখুন — ব্যাকরণ, কানজি, ফুরিগানা, অনুবাদ সবকিছু একসাথে। নিচের যেকোনো একটি প্রশ্ন দিয়ে শুরু করুন।

AI উত্তর সর্বদা সঠিক নাও হতে পারে। গুরুত্বপূর্ণ তথ্য যাচাই করুন।

সেটিংস

আপনার AWS API Gateway এন্ডপয়েন্ট URL দিন। খালি রাখলে ডেমো মোডে চলবে।
API Gateway x-api-key হেডারে ব্যবহৃত হবে।
