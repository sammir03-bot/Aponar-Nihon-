Minna no Nihongo • Lesson 7

で as Tool/Language + あげます・もらいます・もう

কিসে/কোন ভাষায় করি, দেওয়া-নেওয়া, এবং “already” বলা

Grammar শুরুQuiz

আজকের Learning Goal

Tool দিয়ে কাজ বলাはしで食べます — চপস্টিক দিয়ে খাই
Give/receive বোঝাあげます / もらいます
Already বলাもう宿題をしました
Detailed Grammar

সহজ ব্যাখ্যা + structure + example

প্রতিটি pattern আগে structure দেখে, তারপর example মুখে বলুন। Audio button চাপলে Japanese sentence শুনতে পারবেন।

N で Vます
Tool
কোন tool/মাধ্যম দিয়ে কাজ করা হচ্ছে তা で দিয়ে বলা হয়।
Lesson 6-এর で ছিল action place; এখানে で = by/with।
[tool/language] で [verb]ます
はしでご飯を食べます。
চপস্টিক দিয়ে ভাত খাই।
日本語でレポートを書きます。
জাপানি ভাষায় রিপোর্ট লিখি।
鉛筆で名前を書きます。
পেন্সিল দিয়ে নাম লিখি।
Word は Language で 何ですか
Language
কোনো শব্দ অন্য ভাষায় কী, তা জিজ্ঞাসা করতে।
বাংলা/ইংরেজি/জাপানি meaning জানতে খুব useful।
「X」は日本語で何ですか
「Thank you」は日本語で何ですか。
“Thank you” জাপানিতে কী?
「ありがとう」です。
আরিগাতো।
Person に N を あげます
Give
আমি/কেউ কাউকে কিছু দিই।
গ্রহীতা/person সাধারণত に দিয়ে আসে।
[giver] は [person] に [thing] を あげます
私は友達に花をあげます。
আমি বন্ধুকে ফুল দিই।
母にプレゼントをあげました。
মাকে উপহার দিয়েছি।
Person に/から N を もらいます
Receive
কারো কাছ থেকে কিছু পাই/নিই।
source/person に বা から — দুটোই দেখা যায়; から বেশি স্পষ্ট “থেকে”।
[receiver] は [person] に/から [thing] を もらいます
私は先生に本をもらいました。
আমি শিক্ষকের কাছ থেকে বই পেয়েছি।
友達から写真をもらいました。
বন্ধুর কাছ থেকে ছবি পেয়েছি।
もう Vました / まだです
Already
কাজ হয়ে গেছে বলতে もう; এখনো হয়নি হলে まだです।
Question: もう Vましたか。 Answer: はい、もう Vました / いいえ、まだです。
もう [Vました] / まだです
もう宿題をしましたか。
হোমওয়ার্ক করেছেন?
はい、もうしました。
হ্যাঁ, করেছি।
いいえ、まだです。
না, এখনো না।
Speaking Practice

মুখে বলার Practice

এই বাক্যগুলো interview/classroom conversation-এর মতো করে ৩ বার করে পড়ুন।

何でご飯を食べますか。
কী দিয়ে ভাত খান?
はしで食べます。
চপস্টিক দিয়ে খাই।
誕生日に何をもらいましたか。
জন্মদিনে কী পেয়েছেন?
友達に本をもらいました。
বন্ধুর কাছ থেকে বই পেয়েছি।
もう宿題をしましたか。
হোমওয়ার্ক করেছেন?

Common Mistakes

❌ 友達を本をもらいました。
✅ 友達に/から本をもらいました。 — source/person = に/から।
❌ はい、まだしました。
✅ はい、もうしました / いいえ、まだです。
